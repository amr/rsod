What do you want to do?

Create a tunnel

You can create a Reverse SSH tunnel for somebody by uploading their RSOD Request file.

RSOD will allow you to shut down the tunnel any time you ask it to.

Manage active tunnels

Manage active tunnels and close unneeded ones.

Create a Request

Generate a Request for Reverse SSH.

RSOD will generate a file with your host information, to facilitate its communication.

RSOD Tunnel Creation form support this file; Thus, you can hand a request to someone who will feed it to RSOD on their machine and a tunnel will be created with the relevant information. Useful when working with less technical people.
